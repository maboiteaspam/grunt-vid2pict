# demo grunt-vid2pict

Demo to shows how thqt works.

## Running

```sh
npm i -g nws
npm i 
bower i
```

Don t forget to update the `Gruntfile.js` to specify `ffmpeg` path 
if it s not already in the path.

Then run `grunt` default task.

```sh
grunt
```

Then starts a webserver

```sh
nws -d web -p 8080 -o
```
